As a developer at Scout, I built a website for INSHORE (International Network for the Study of Rocky Intertidal Ecosystems), a group of scientists dedicated to climate change research through the lens of marine sustainability.

After making a static HTML prototype, I built a custom Wordpress theme in PHP and continued developing on a live staging server so that both the Scout team and the client could see my progress.

The most challenging part was creating the map on the About page. I initially tried to use jQuery with the Google Maps API, but because it had to dynamically sync with the MySQL database, I ended up using a modified version of a plugin called GeoMashup. (However, this jQuery knowledge came in handy at Cal Hacks)

See the site live at rockyinshore.org!
